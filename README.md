# worktime
労働時間に関する計算を行う

## Install
利用するプロジェクトの `composer.json` に以下を追加する。
```composer.json
"repositories": {
    "period": {
        "type": "vcs",
        "url": "https://github.com/shimoning/worktime.git"
    }
},
```

その後以下でインストールする。

```bash
composer require shimoning/worktime
```
